mysql

mysql的打开：找到放mysql的文件，找到他的bin文件，我的是在e盘的E:\phpstudy_pro里（小皮的）
然后在那里打开cmd命令，通过mysql -u root -p 其中u后面的是用户名
输入密码一般是root

mysql基本命令
show databases; //查看所有数据库
crete database study; //创建study数据库
use study； //进入study数据库
show tables; //查看所有数据表

int表示该段为整型，其中 auto….表示一个自动增长的段， prim….表示主键
var…..表示限长为50 not null表不能为空

select *from user //查看表中的所有内容
INSERT INTO users (username, password) VALUES (‘zhangsan’, ‘123456’); 在表中插入内容，两个字段要与定义表时一样。（看他允不允许空）

UPDATE users SET username = 'admin', password = 'admin' WHERE username = 'zhangsan';
修改根据where后面的字段来寻找并且更新表的内容

DELETE FROM users WHERE WHERE username = ‘zhangsan’; //删除users表中的内容通过where后面的判断，提示如果没有判断将会把此表中的所有内容全部删除。

drop table users; //删除users表
drop database study; //删除数据库
